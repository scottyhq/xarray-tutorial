Working with Multiple Datasets#

Learning goals:

  • Perform operations across multiple datasets

import numpy as np
import xarray as xr

np.random.seed(0)

Broadcasting and automatic alignment#

Broadcasting refers to the alignment of arrays with different numbers of dimensions. Numpy’s broadcasting rules, based on array shape, can sometimes be difficult to understand and remember. Xarray does broadcasting by dimension name, rather than array shape. This is a huge convenience.

arr1 = xr.DataArray(
    np.random.randn(3, 4),
    dims=("x", "y"),
    coords={"x": ["a", "b", "c"], "y": np.arange(4)},
)
arr2 = xr.DataArray(
    np.random.randn(2, 7),
    dims=("x", "y"),
    coords={"x": ["b", "d"], "y": [-2, -1, 0, 1, 2, 3, 4]},
)

arr1 + arr2

This will automatically select only common labels from both arrays (a inner join) and then perform the operation.

Warning: If the arrays being broadcasted share a dimension name, but have different coordinates, they will first be aligned using Xarray's default align settings (including filling missing values with NaNs). If that's not what you want, it's best to call align explicitly before broadcasting.

A More Detailed Look#

What Xarray is doing automatically to perform the addition operation is a combination of broadcast and align.

First, both arrays were automatically broadcasted against each other:

arr1_, arr2_ = xr.broadcast(arr1, arr2)
